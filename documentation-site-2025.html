Last updated: January 2025

Author: Design Team

What is the 2025 Design System?

The 2025 Design System is a comprehensive collection of design principles, components, and guidelines that ensure consistency, accessibility, and modern user experience across the Angkor Compliance Platform. It incorporates the latest design trends including glassmorphism, neumorphism, and bento grids while maintaining professional standards for compliance and audit environments.

Key Features

  • Role-Based Theming: Different visual themes for various user roles (Worker, Factory Admin, HR Staff, etc.)
  • Modern Design Patterns: Glassmorphism, neumorphism, and bento grid layouts
  • Accessibility First: WCAG 2.1 AA compliance with comprehensive accessibility features
  • Responsive Design: Mobile-first approach with comprehensive breakpoints
  • Dark Mode Support: Automatic and manual dark mode with system preference detection
  • Internationalization: Support for Khmer and English text with proper typography
  • Performance Optimized: Critical CSS, lazy loading, and optimized component rendering
  • PWA Ready: Progressive Web App features with offline support

Design Principles

Our design principles guide every decision in the 2025 Design System, ensuring consistency, accessibility, and user-centered design.

1. Accessibility First

Every component is designed with accessibility in mind, meeting WCAG 2.1 AA standards. This includes proper color contrast, keyboard navigation, screen reader support, and focus management.

2. Consistency

Consistent design patterns, spacing, typography, and interactions create a cohesive user experience across all parts of the platform.

3. Role-Based Design

Different user roles have distinct visual themes and interaction patterns that support their specific workflows and responsibilities.

4. Modern & Professional

We incorporate modern design trends like glassmorphism and neumorphism while maintaining the professional appearance required for compliance and audit environments.

5. Performance

All components are optimized for performance with critical CSS, lazy loading, and efficient rendering.

6. Internationalization

The system supports both Khmer and English text with proper typography and layout considerations.
